### Day3（step_order 4）※テーマ別ミニ講義＝なぜ息切れ・むくみが起こるのか（前負荷・後負荷の超入門）
```
今日は心不全の“土台”をひとつ。なぜ息切れ・むくみが起こるのか、ざっくりいきましょう！（教育目的）

Q. 心不全で息切れやむくみが出るのは、なぜ？
　A. 血液が足りなくて、体が乾いてしまうから
　B. 血液をうまく前へ送れず、“渋滞”して水があふれるから

……考えてみてください🤔

―――――――――――
答え：B・渋滞です！
心不全は、ポンプ（心臓）の力が落ちて、血液をうまく前へ送り出せない状態。すると血液は上流で“渋滞”します。
・肺の側で渋滞 → 肺に水がにじむ → 息切れ
・全身（静脈）の側で渋滞 → 足などに水がたまる → むくみ

循環器ルートで出てきた「前負荷＝心臓に入ってくる量」を思い出してください。戻ってきた血液をさばききれず溜まっていく——それが“渋滞”の正体です。

「心不全＝難しい病気」と身構えなくて大丈夫。まずは「体の中で水の流れが渋滞している」とイメージできれば十分です☺️

beナビの勉強会は、いつもこんな空気です。
※教育目的。診断・治療は医師の判断に従ってください。
```
