### Day2（step_order 3）
```
なぜ、beナビをつくったのか。

僕は看護師として20年、心臓カテーテル室で現場と向き合ってきました。
現場で痛感したのは——心臓デバイスの知識が「ある」人は増えても、「使える・語れる」人は驚くほど少ないこと。そして何より、みんな一人で、孤独に学んでいたことです。

参考書は教えてくれる。でも「これで合ってる？」を聞ける仲間はいない。続かない。もったいない。

だから決めました。ナースが心臓デバイスを通してつながり、語り合える場所をつくろう、と。それがbe Navigatorです。

ここは“サービス”じゃなく“仲間の集まり”。あなたが入ってくれたことで、また一つ、心臓デバイスを語れる輪が広がりました。本当にありがとう。
——ベッカミ
```

### Day3（step_order 4）※テーマ別ミニ講義＝ペースメーカーの基本（なぜ必要か／何を見るか）
```
今日は心臓デバイスの“土台”をひとつ。ペースメーカー、怖がらずにいきましょう！（教育目的）

Q. ペースメーカーって、どんな役割だと思いますか？
　A. 心臓を動かしている主役
　B. 遅くなりすぎたリズムを支える“補助”

……考えてみてください🤔

―――――――――――
答え：B・補助です！
心臓を動かしている主役は、あくまであなた自身の心臓。ペースメーカーは「心臓を動かす機械」ではなく、実際は、脈が遅くなりすぎたとき（徐脈）だけ支える補助役です。自分の脈（自己脈）が出ているときは、基本それを見守っています。

だから心電図を見るときは、難しい設定やモードの暗記より、まずこの4つをざっくり：
①なぜ入っているか（多くは「脈が遅すぎる」を支えるため）
②自己脈は出ているか
③ペーシング波形（スパイク）が出ているか
④症状と脈拍が、つじつま合っているか

この4つが分かれば、「あ、ちゃんと仕事してるな」が見えてきます。まずはこの4つだけ見られれば十分です。怖がらなくて大丈夫☺️

beナビの勉強会は、いつもこんな空気です。
※教育目的。診断・治療は医師の判断に従ってください。
```
