# beナビ 7日オンボーディング 共通テンプレート（教育型ルート用）

> 教育型ルート（device / heartfailure / ai など）の7日オンボを**量産するための雛形**。
> circ（[benavi-circulation-scenario.md](benavi-circulation-scenario.md)）で固まった共通構造を抽出したもの。
> **変わるのは実質 Day3・Day5 の2枠だけ**。Day0/1/2/6/7 は「テーマ名（`{{route_theme}}`）」を差し替えるだけで成立するよう一般化してある。
>
> ※ goods / event は「情報フィード型」で構造が別。本テンプレは**教育型7日**専用。

## 使い方（新ルート量産の最短手順）

1. このテンプレを複製し、`benavi-<route>-scenario.md` を作る。
2. **`{{route_theme}}`**（テーマ名）を一括置換（例：`循環器` / `心臓デバイス・ペースメーカー` / `心不全` / `AI活用`）。
3. **Day3** に「テーマ別ミニ講義（クイズ）」を執筆（唯一の必須執筆①）。
4. **Day5** の `{{day5_theme_content}}` に「テーマ別コンテンツ紹介」を執筆（唯一の必須執筆②）。
5. 固定UUIDを発番（scenario 1 + steps 7）。`trigger_tag_id` は当該 `R:<route>` タグID。
6. INSERT用 `.sql` を生成（Codex委譲推奨）→ Claudeレビュー → 適用は別途オーナーGo。
7. ミラーmd冒頭に「案・未適用」を付けて作り、適用後に circ と同様「本番稼働中」へ更新。

> **執筆が要るのは Day3 と Day5 の2枠だけ**。他5日はテーマ名置換のみ。

## シナリオ定義（テンプレ）

| 項目 | 値 |
|---|---|
| trigger_type | `tag_added` |
| trigger_tag_id | 当該 `R:<route>` タグID（[V1_1_implementation_plan.md](beNavigator_CRM_V1_1_implementation_plan.md) のタグ表参照）|
| delivery_mode | `relative` |
| is_active | 1 |
| 診断分岐 | **なし**（`{{type_tip}}` / `diagnosis_label` は使わない。ecg専用機能） |
| message_type | 全ステップ `text` |

## 配信タイミング（全教育型ルート共通）

| step_order | 役割 | delay_minutes（前stepから） | 可変？ |
|---|---|---|---|
| 1 | Day0 乗船・歓迎・仲間づくり | 0 | テーマ名のみ |
| 2 | Day1 コミュニティ紹介 | 1440 | テーマ名のみ |
| 3 | Day2 be Navigatorの理念 | 1440 | テーマ名のみ |
| 4 | **Day3 テーマ別ミニ講義** | 1440 | **★全文を執筆** |
| 5 | **Day5 テーマ別コンテンツ紹介** | 2880 | **★中段を執筆** |
| 6 | Day6 参加者の変化・安心感 | 1440 | テーマ名のみ |
| 7 | Day7 自然なCTA | 1440 | テーマ名のみ |

> Day4 は枠なし（Day3→Day5 は 2880分＝2日間隔）。delay 並び＝`0/1440/1440/1440/2880/1440/1440`。

# 可変2枠（★ここだけルートごとに執筆）

## Day3（step_order 4）— テーマ別ミニ講義

**型**（circ Day3 と同じ流れ）：
```
今日は{{route_theme}}の“土台”をひとつ。〇〇、ざっくりいきましょう！（教育目的）

Q.〔初心者向けに二択推奨〕
　A. 〔選択肢A〕
　B. 〔選択肢B〕

……考えてみてください🤔

―――――――――――
答え：〔正解〕！
・〔ざっくり解説1〕
・〔ざっくり解説2〕

〔この2語/この考え方が分かると、現場の何が読み解きやすくなるかを一言〕

beナビの勉強会は、いつもこんな空気です☺️
※教育目的。診断・治療は医師の判断に従ってください。
```
- 初心者ルートなので**二択クイズ推奨**（心理的ハードルを下げる）。
- circ実例 = 「前負荷・後負荷の超ざっくり理解」（[benavi-circulation-scenario.md](benavi-circulation-scenario.md) の Day3 を参照）。

## Day5（step_order 5）— テーマ別コンテンツ紹介

**型**（前後は共通の予告トーン。中段 `{{day5_theme_content}}` だけ執筆）：
```
少しずつ、この船のことが分かってきてくれたでしょうか😊
今日は、船のいちばん奥——正式なクルーだけが集まる、特別な甲板のお話を。

{{day5_theme_content}}
さらに、毎晩21:30からの勉強会（週5回）と、そのアーカイブ236本以上が、クルーになれば好きな時に好きなだけ見放題。「今日のあの患者さん」を持ち帰って、翌日には仲間と語れる——そんな学び方ができます。

ここは、クルーになると開く“港”。同じ海をめざす仲間が、全国で待っています。
いつか、この灯りの中で、あなたと話せる日を楽しみにしています。
——ベッカミ
```
- `{{day5_theme_content}}` 例（circ）：「beナビには、{{route_theme}}を体系的に学べる勉強会やコンテンツがあります。〔土台〕から〔発展〕まで、現場でつながる形で一歩ずつ。」
- **実態に無いコンテンツ名を断定しない**（circでは固定名称「循環器シリーズ」を避け「勉強会やコンテンツ」とした）。週5回/21:30/アーカイブ236本以上は実態どおりなので共通枠に固定。
