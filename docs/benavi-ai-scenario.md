### Day2（step_order 3）
```
なぜ、beナビをつくったのか。

僕は看護師として20年、心臓カテーテル室で現場と向き合ってきました。
現場で痛感したのは——AI活用の知識が「ある」人は増えても、「使える・語れる」人は驚くほど少ないこと。そして何より、みんな一人で、孤独に学んでいたことです。

参考書は教えてくれる。でも「これで合ってる？」を聞ける仲間はいない。続かない。もったいない。

だから決めました。ナースがAI活用を通してつながり、語り合える場所をつくろう、と。それがbe Navigatorです。

ここは“サービス”じゃなく“仲間の集まり”。あなたが入ってくれたことで、また一つ、AI活用を語れる輪が広がりました。本当にありがとう。
——ベッカミ
```

### Day3（step_order 4）※テーマ別ミニ講義＝現場で使えるAIの“安全な”使いどころ
```
今日はAI活用の“土台”をひとつ。AIの「安全な使いどころ」、ざっくりいきましょう！（教育目的）

Q. 看護の現場でAIを使うとき、いちばん大事な“原則”はどっち？
　A. AIが出した答えを、そのまま正解として使う
　B. AIはあくまで“補助”、最終判断は必ず人がする

……考えてみてください🤔

―――――――――――
答え：B・最終判断は人、です！
AIは「魔法の道具」でも「診断してくれる人」でもありません。得意なのは、こんな“補助”の場面👇
・考えごとの整理（「この理解で合ってる？」の壁打ち相手）
・文章を整える（申し送りや勉強メモを読みやすく）
・学びのきっかけ（わからない言葉を、まずざっくり聞いてみる）

逆に、ここだけは守ってほしい2つ：
①患者さんの情報や個人情報、勤務先の機密情報は入力しない（情報が外に出るリスク）
②医療判断や診断そのものは、AIに丸投げしない——最後は必ず人が確かめる

この2つさえ守れば、AIは怖い道具じゃなく、心強い“学びの相棒”になります☺️

beナビの勉強会は、いつもこんな空気です。
※教育目的。診断・治療は医師の判断に従ってください。
```
